# 📊 E-Commerce Sales Dashboard (Power BI)

## 🚀 Project Overview
This project presents an interactive E-Commerce Sales Dashboard built using Power BI, designed to analyze and visualize key business metrics from a dataset of 160,000+ rows.

The goal of this project is to transform raw transactional data into meaningful insights that support data-driven decision-making.

---

## 📁 Dataset Information
- Total Records: 160,000+ rows
- Data Type: E-commerce transactional data  
- Includes: Orders, Products, Users, Revenue, Categories, Time  

---

## 📊 Key Features
- Revenue Analysis Dashboard  
- Conversion Funnel (View → Cart → Purchase)  
- User Distribution (App vs Browser)  
- Category & Sub-Category Performance  
- Top 5 Best-Selling Products  
- Peak Sales Time Analysis  
- Quarterly & Monthly Trends  
- Interactive Filters (Brand, Category, Sub-Category)

---

## 🛠 Tools & Technologies Used
- Power BI  
- DAX (Data Analysis Expressions)  
- Data Cleaning & Transformation  
- Data Modeling  
- Data Visualization  

---

## 📈 Key Insights
- Total Revenue reached 5.65M+  
- Peak sales occur between 6 AM – 6 PM  
- Smartphones are the most demanded product category  
- Conversion Rate: 11.44% (optimization opportunity)  
- Highest revenue observed in Q4  

---

## 🎯 Objectives
- Analyze large-scale e-commerce data (160K+ rows)  
- Identify trends in customer behavior  
- Track KPIs like revenue, conversion rate, and user activity  
- Build a visually appealing and interactive dashboard
